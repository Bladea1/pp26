# Лаборатория недоделанных проектов — V3 Canonical

Новая пересборка под утверждённые макеты: чёрный cyber-grunge интерфейс, кислотный лайм, hot pink, плотная сетка, лабораторные карточки и механика «образец → исходники → реактив → мутация → фаза».

## Запуск

```bash
npm install
npm run dev
```

Открыть: http://localhost:3000

## Supabase (обязательно)

1. Выполните в SQL Editor: `schema.sql` → `auth.sql` → `writes.sql` → `seed.sql`
2. Заполните `.env.local` (см. `.env.example` и `docs/SUPABASE.md`)

Без базы сайт откроется, но каталог, журнал и профиль будут пустыми. Создание образцов и реактивов требует входа в аккаунт.
